# 캐릭터 입체성 채점 기준표 (Character Embodiment Rubric)

quality-verifier와 character-sculptor가 캐릭터 입체성을 점수화할 때 사용하는 **0-10점 채점 기준표**.

---

## 7개 차원별 채점 기준

### 1. VOICE (대사 고유성) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 대부분의 대사가 다른 캐릭터와 교환 가능. 종결어미/말투가 캐릭터시트와 불일치 |
| 3-4 | 종결어미는 일치하나, 사고 패턴·관점이 일반적. 교환 가능 대사 50%+ |
| 5-6 | 캐릭터 고유 말투가 일관되나, 정보 전달 대사에서 DNA가 빠짐. 교환 가능 대사 20-50% |
| 7-8 | 대부분의 대사에서 캐릭터 DNA가 느껴짐. 교환 가능 대사 10% 미만. 상황별 변주 있음 |
| 9-10 | 모든 대사가 이 캐릭터만의 사고 패턴을 반영. 화자 표시 없이도 누구인지 알 수 있음 |

### 2. NONVERBAL (비언어 서술) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 비언어 없거나 "그는 웃었다" 수준의 태그 직결만 존재 |
| 3-4 | 비언어 있으나 기계적 삽입(태그 직결 3건+). 인과 순서 미준수 |
| 5-6 | 인과 순서(자극→신체→행동→대사) 부분 준수. 같은 비언어 3회+ 반복 |
| 7-8 | 인과 순서 일관 준수. 캐릭터별 고유 비언어 사용. 강도가 사건에 비례. 반복 2회 이하 |
| 9-10 | 비언어가 서술에 완전히 녹아있음. 5단계 강도 변주 자연스러움. 3화간 반복 없음 |

### 3. INNER (내면 반영) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 캐릭터 내면(욕망/공포/모순)이 전혀 드러나지 않음. 기능적 존재 |
| 3-4 | 설정상 동기가 언급되나, 행동과 연결 없음 |
| 5-6 | 동기가 행동의 일부에 반영됨. 내면 갈등이 간접적으로 보임 |
| 7-8 | 행동·선택·침묵에서 내면이 일관되게 드러남. 모순이 자연스러운 긴장으로 작용 |
| 9-10 | 매 장면에서 내면이 행동의 동력이 됨. 독자가 캐릭터의 다음 행동을 예측하면서도 놀랄 수 있음 |

### 4. RELATION (관계 역동) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 캐릭터 간 관계가 평면적. 모든 상대에게 동일하게 반응 |
| 3-4 | 관계 설정은 있으나 대화/행동에 반영 안 됨 |
| 5-6 | 주요 관계에서 거리감/친밀감이 부분적으로 반영됨 |
| 7-8 | 관계 단계에 따른 호칭·톤·거리감이 일관됨. alive-tracker 아크 단계와 정합 |
| 9-10 | 관계의 미묘한 변화가 비언어·대사·행동에 자연스럽게 스며들어 있음. 독자가 관계 변화를 '느낌' |

### 5. ASTONISHMENT (경악 고유성) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 모든 캐릭터가 동일하게 놀람 ("눈이 커졌다", "입이 벌어졌다"만 사용) |
| 3-4 | 캐릭터별 경악 방식이 있으나 강도 변주 없음 (항상 같은 반응) |
| 5-6 | 경악 방식은 고유하나, 이벤트 무게와 비례하지 않음 |
| 7-8 | 캐릭터별 고유 경악 방식 + 사건 무게에 따른 5단계 강도 변주. 같은 경악 화당 2회 이하 |
| 9-10 | 경악이 캐릭터의 정체성을 드러내는 도구로 작용. 압도(5단계) 시 대비 효과가 극적 |

### 6. CATHARSIS (카타르시스 기여) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 카타르시스 장면에 기여 없음. 배경 인물 수준 |
| 3-4 | 카타르시스 장면에 존재하나, 반응이 형식적 |
| 5-6 | 주인공의 카타르시스를 부분적으로 증폭하는 역할 |
| 7-8 | 카타르시스 장면에서 고유한 반응으로 독자 감정을 증폭. 사이다 장면의 관중 역할 수행 |
| 9-10 | 이 캐릭터의 반응 자체가 카타르시스의 일부. 독자가 이 캐릭터의 시점에서 함께 느낌 |

### 7. DIALOGUE DNA (대화 사고 패턴) — 0~10점

| 점수 | 기준 |
|------|------|
| 0-2 | 대사가 정보 전달만 수행. 사고 패턴 없음 |
| 3-4 | 말투는 있으나, 정보를 처리하는 방식(분석형/직관형/감정형 등)이 일반적 |
| 5-6 | 캐릭터 DNA가 부분적으로 반영되나, 기능 대사에서 빠짐. 같은 사고경로 3회+ 반복 |
| 7-8 | 모든 상황 유형(좋은 소식/위기/갈등)에서 고유 사고경로를 따름. 교체 불가성 90%+ |
| 9-10 | 대사만으로 캐릭터를 식별 가능. 상황별 변주가 자연스럽고, 성장에 따른 변화도 반영 |

---

## 종합 점수 산출

### 주인공/핵심 캐릭터 (매 EP 등장)
- 7개 차원 전체 평가
- **종합 점수 = 7개 차원 평균**
- PASS 기준: 평균 7.0점 이상, 개별 차원 5.0점 이상

### 주요 조연 (해당 EP에 5문장+ 등장)
- 평가 차원: VOICE, NONVERBAL, RELATION, DIALOGUE DNA (4개)
- **종합 점수 = 4개 차원 평균**
- PASS 기준: 평균 6.0점 이상

### 단역/배경 (해당 EP에 1-4문장 등장)
- 평가 차원: VOICE (1개)
- **종합 점수 = VOICE 점수**
- PASS 기준: 5.0점 이상

---

## REWRITE 판정 연동

quality-verifier REWRITE 모드에서:
- **캐릭터 입체성 평균 7점 이상** → PASS 조건 충족
- **캐릭터 입체성 평균 5점 미만** → 즉시 REWRITE
- **5~7점 사이** → 다른 축과 종합하여 판정 (개별 차원 3점 이하가 2개+ → REWRITE)
